### How to use it

Clone the repository `[somewhere]`

    # Make .bashrc load what we want
    echo ". [somewhere]/bashrc" > ~/.bashrc
    # Link git config
    ln -sf [somewhere]/gitconfig ~/.gitconfig
    # Link mercurial config
    ln -sf [somewhere]/hgrc ~/.hgrc
